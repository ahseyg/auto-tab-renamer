## Overview

**Do you have too many tabs open?** Do you struggle to identify tabs because the titles are generic or truncated?
**Auto Tab Renamer** solves this problem by automatically renaming your tabs based on the page content. It ensures you can always identify your tabs at a glance, improving your workflow and organization.

## Features

*   **Dynamic Renaming:** Automatically updates tab titles based on the DOM content.
*   **CSS Selectors:** Extract titles from any element using standard CSS selectors.
*   **Customizable Rules:** Create specific rules for any website using Regex URL matching.
*   **Auto-Fill:** Quickly generate rules from the current tab with a single click.
*   **Built-in Support:** Pre-configured rules for ChatGPT and Google Gemini.
*   **Vertical Tab Optimization:** Optimized for browsers like Zen Browser with auto-truncation features.

## Installation

### Chrome / Edge / Brave

1.  Download or clone this repository.
2.  Navigate to `chrome://extensions/`.
3.  Enable "Developer mode" in the top right corner.
4.  Click "Load unpacked" and select the `auto-tab-renamer` root directory.

> **ℹ️ Coming soon to Chrome Web Store**

### Firefox / Zen Browser

**Installation:**
1.  Navigate to `about:debugging#/runtime/this-firefox`.
2.  Click "Load Temporary Add-on...".
3.  Select the `manifest.json` file.

> **ℹ️ Coming soon to Firefox Add-ons**

## Usage

1.  Click the extension icon in your toolbar.
2.  Select **"Add Rule"**.
3.  Use the **"Auto-Fill"** button to populate the URL pattern.
4.  Enter the valid **CSS Selector** for the target title element.
5.  Save the rule.

## Genel Bakış

**Çok fazla sekmeniz mi açık?** Başlıklar yetersiz olduğu veya kesildiği için sekmelerinizi ayırt etmekte zorlanıyor musunuz?

## Kurulum

### Chrome / Edge / Brave

1.  Bu depoyu indirin veya klonlayın.
2.  `chrome://extensions/` adresine gidin.
3.  Sağ üst köşedeki "Geliştirici modu"nu etkinleştirin.
4.  "Paketlenmemiş öğe yükle" seçeneğine tıklayın ve `auto-tab-renamer` ana klasörünü seçin.

> **ℹ️ Yakında Chrome Web Mağazası'nda**

### Firefox / Zen Browser

**Kurulum:**
1.  `about:debugging#/runtime/this-firefox` adresine gidin.
2.  "Geçici Eklenti Yükle..." düğmesine tıklayın.
3.  `manifest.json` dosyasını seçin.

> **ℹ️ Yakında Firefox Eklentileri'nde**

## Kullanım

1.  Araç çubuğundaki eklenti simgesine tıklayın.
2.  **"Add Rule"** (Kural Ekle) seçeneğini seçin.
3.  URL desenini otomatik doldurmak için **"Auto-Fill"** (Otomatik Doldur) düğmesini kullanın.
4.  Hedef başlık elementi için geçerli **CSS Seçicisini** girin.
5.  Kuralı kaydedin.
